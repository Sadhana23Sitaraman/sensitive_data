# AI Sensitive Data Filter (Python, Regex, Security Automation)

A regex-based sensitive data detection and redaction module built for security automation.

### Features
- Detects PII (emails, phone numbers, SSN)
- Detects credentials (username:password)
- Detects API keys (AWS keys, generic tokens)
- Detects JWT tokens, PEM private keys
- Shannon entropy analysis for high-entropy secrets
- Redaction with deterministic masking
- CLI support
